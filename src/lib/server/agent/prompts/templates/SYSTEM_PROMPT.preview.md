# System Prompt Preview

This file is a static placeholder only.

The live system prompt preview is generated by `buildSystemPromptPreview()` in `src/lib/server/agent/prompts/prompt.ts`.
Use the Web Chat prompt preview endpoint or the runtime-generated `SYSTEM_PROMPT.preview.md` in the active bot workspace when auditing the actual prompt sent to the model.

Do not copy old event, skill, sandbox, or tool-routing examples into this file. Scheduling and reminders must route through the deferred `createEvent` tool after loading it with `toolSearch`; they must not be implemented by manually writing watched event JSON files.

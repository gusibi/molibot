# Momo for Mac — 设计规范

> macOS 26「液态玻璃 / Liquid Glass」风格的桌面客户端。当前覆盖 **Chat 页** 与 **Settings 页**。
> 源文件：`Momo for Mac.dc.html`；独立离线版：`Momo for Mac (standalone).html`。

---

## 1. 设计原则

1. **液态玻璃**：所有窗口/侧栏为半透明磨砂面板，背后壁纸透出。靠 `backdrop-filter: blur + saturate` + 顶部高光 + 极细白色描边实现。
2. **浅色优先**：以浅色为默认主题（参考浅色稿），白/灰带极低饱和度，背景壁纸提供色彩。
3. **遵循原生**：Chat 参考 macOS Messages，Settings 严格参考 macOS 系统设置（彩色圆角图标侧栏 + 分组圆角卡片行）。
4. **克制**：不堆砌图标和数字；留白与层级营造节奏，1–2 种背景色为限。

## 2. 颜色

### 主色 / 强调色（可切换）
| 名称 | 值 | 用途 |
|---|---|---|
| 系统蓝（默认） | `#007AFF` | 用户气泡、发送按钮、开关开启、链接、选中态 |
| 备选强调色 | `#AF52DE` 紫 / `#FF2D55` 粉 / `#FF9500` 橙 / `#34C759` 绿 / `#8E8E93` 石墨 | 强调色可选项 |

> 强调色统一以 CSS 变量 `--accent` 注入，子元素用 `var(--accent, #007AFF)`；浅色衬底用 `--accent-soft`（强调色 + 14% 透明，即 `#007AFF24`）。

### 中性色
| 角色 | 值 |
|---|---|
| 主文字 | `#1c1c1e` |
| 次级文字 | `rgba(60,60,67,0.6)` |
| 三级 / 占位 | `rgba(60,60,67,0.45)` |
| 分隔线 | `rgba(0,0,0,0.06~0.07)` |
| 中性填充（搜索框/未选 chip/图标底） | `rgba(120,120,128,0.12~0.16)` |

### 状态色
- 已连接 / 在线：`#34C759`
- 进行中：`var(--accent)`
- 未连接 / 等待：灰 `rgba(60,60,67,0.5)` / `rgba(60,60,67,0.28)`

### 渠道色
网页/Telegram `#2AABEE` · 飞书 `#3370FF` · QQ `#12B7F5` · 微信 `#07C160` · 本地 Momo `#1c1c1e`

### 壁纸（玻璃背景，可切换）
- **晨曦**（默认）：蓝→薰衣草→桃 渐变 + 角向径向光
- **云海**：清透蓝白
- **薄暮**：靛紫
- **石墨**：中性灰

## 3. 液态玻璃 Token

| Token | 浅色默认值 | 减少透明度时 |
|---|---|---|
| 窗口面板 `--panel-bg` | `rgba(250,250,252,0.7)` | `rgba(244,244,247,0.98)` |
| 侧栏 `--sidebar-bg` | `rgba(255,255,255,0.4)` | `rgba(236,236,240,0.99)` |
| 卡片 `--card-bg` | `rgba(255,255,255,0.72)` | `#ffffff` |
| 模糊 `--blur` | `blur(40px) saturate(180%)` | `saturate(120%)` |

**窗口外壳**
```
border-radius: 20px;
border: 0.5px solid rgba(255,255,255,0.7);
box-shadow: 0 40px 80px -24px rgba(28,38,68,0.5),
            0 2px 10px rgba(0,0,0,0.08),
            inset 0 1px 0 rgba(255,255,255,0.5);   /* 顶部高光 */
```
**气泡 / 卡片**：白色半透 `rgba(255,255,255,0.78)` + `0.5px` 描边 + 柔和投影 `0 2px 8px -3px rgba(0,0,0,0.12)`。

## 5. 布局

整体为一张 macOS 桌面场景（`1920×1130`）：顶部菜单栏 + Chat 窗口（左）+ Settings 窗口（右），均浮在壁纸上。

### Chat 窗口（1052 宽，三栏）
```
┌── 侧栏 270 ──┬──── 对话流 flex ────┬── 文件面板 288 ──┐
│ 品牌/新对话   │ 头部(头像+状态+操作) │ 文件标题/筛选     │
│ 自动任务…     │ 消息流              │ 按日期分组的文件   │
│ 本地知识库     │ 输入框 / 只读提示    │ 存储用量          │
│ 对话(按渠道分组)│                    │                  │
│ 账户与设置     │                    │                  │
└──────────────┴────────────────────┴─────────────────┘
```
- **文件面板**可经头部按钮收起/展开；收起时对话流占满。

### Settings 窗口（744 宽，两栏，原生系统设置）
```
┌── 侧栏 228 ──┬──────── 内容 flex ────────┐
│ 搜索          │ 标题(通用)                │
│ 彩色图标分类   │ 分组圆角卡片：语言/外观/   │
│ 账户与设置     │ 通知/启动与显示/渠道连接/  │
│              │ 数据与存储                │
└──────────────┴──────────────────────────┘
```

## 6. 组件规范

- **窗口红绿灯**：12px 圆点，`#ff5f57 / #febc2e / #28c840`，位于窗口左上、侧栏内。
- **侧栏列表项**：高 32–46px，圆角 8–9px，左侧 16–30px 圆角图标，选中态 = `--accent-soft` 底 + 强调色文字/图标。
- **新对话按钮**：强调色浅底胶囊 + 强调色文字。
- **消息气泡**：用户=强调色实底白字，右对齐，圆角 `18 18 6 18`；助手=白玻璃左对齐，圆角 `18 18 18 6`，左侧 30px 头像。
- **附件卡 / 任务进度卡 / 待办卡**：白玻璃圆角卡，彩色类型图标 + 标题 + 元信息；进度用 ✓(绿) / 旋转圈(强调色) / 空心圈(灰)。
- **输入框**：玻璃胶囊，占位文字在上、左侧附件/应用/语音图标 + 右侧强调色圆形发送按钮。
- **开关 Toggle**：38×22 胶囊，开=强调色 + 滑块右移，关=灰；可点击翻转。
- **系统设置行**：分组卡片内多行，行高 ~46px，行间 `0.5px` 分隔线；右侧控件为弹出按钮（值 + ▾）、开关、或值 + 右箭头 `›`。
- **分类图标**：22px 圆角方块 + 白色实心 Phosphor 图标，每类一色（通用灰/模型紫/渠道蓝/工具橙/记忆粉/隐私青/关于灰）。

## 7. 多渠道与权限模型 ★

- 对话**按渠道分组**：`Momo · 本地`、`飞书`、`Telegram`、`微信`、(`QQ` 未连接)。组头含渠道图标 + 数量，可折叠。
- **本地（Momo）渠道 = 可输入可编辑**：完整 Agent 对话流 + 输入框。
- **其它渠道 = 只读**：
  - 展示该渠道的对话记录（含对方头像/昵称的转写）。
  - 头部显示「通过 <渠道> · 仅查看」+ 渠道色徽章。
  - 底部不显示输入框，改为只读提示条：「此对话来自 X 渠道，在 Momo 中仅支持查看 · 前往 X 回复」。
  - 组头带 👁 标记。

---

## 8. 文件管理面板

- 管理对话中产生的文件，按 `今天 / 昨天 / 更早` 分组。
- 每项：彩色类型图标（docx 蓝 / png 青 / xlsx 绿 / pdf 红 / md 灰）+ 文件名 + `大小 · 来源会话` + 更多操作。
- 顶部筛选 chip：全部 / 文档 / 图片 / 表格；底部显示存储用量与「管理」。

---

## 9. 可调项（Tweaks）

`Momo for Mac.dc.html` 暴露三个属性：
- **accentColor** — 强调色（6 个 macOS 系统色）
- **wallpaper** — 壁纸（晨曦/云海/薄暮/石墨）
- **reduceTransparency** — 减少透明度（关闭玻璃模糊，面板转不透明，对应 macOS 辅助功能）

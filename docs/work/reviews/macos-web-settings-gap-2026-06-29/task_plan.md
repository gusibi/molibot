# macOS App / Web 设置功能差异盘点

## 目标

基于源码比较 Web 设置页与 macOS App 的真实能力，产出可用于后续开发排期的差异清单；本轮不修改业务代码。

## 阶段

1. [complete] 建立设置模块清单与代码入口
2. [complete] 对比 AI 供应商与模型管理
3. [complete] 对比其余设置模块及保存/测试能力
4. [complete] 交叉核对桌面 API、类型与 UI 接线情况
5. [complete] 输出分级差异报告并验证证据链接

## 成功标准

- Web 与 macOS App 的设置模块均有完整清单。
- 每项差异标注为：一致、App 部分支持、App UI 未接 API、App 完全缺失、行为不一致。
- 重点覆盖供应商、模型类型、文本/图片/语音/视频能力及保存和测试流程。
- 每项关键结论可追溯到源码文件。

## 约束

- 只读分析业务代码。
- 不运行或截图 macOS App。
- 不覆盖当前工作树中已有的未提交改动。
